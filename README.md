# PotionGames
> A Minecraft Minigames Plugin for Bukkit Servers!

PotionGames is a mingames plugin that works like SurvivalGames but with Potions and Effects

![](header.png)

## Installation

OS X & Linux:

```sh
npm install my-crazy-module --save
```

Windows:

```sh
edit autoexec.bat
```
